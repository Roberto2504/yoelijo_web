Maes la vara esta asi, en este proyecto si vamos a trabajar lo mas ordenado posible.


Para empezar a bretear tienen que clonar el proyecto y jalar develop (git fetch, git checkout develop, git pull origin develop)


Se crean su branch


Es importante que tengan instalada la version de node mas reciente el proyecto esta montado con la 8.1.3


Le dan 'npm install'


Luego 'npm run dev' y listo tiene el proyecto corriendo en el localhost:2504 (2504  mi fecha de nacimiento puuutos)


Una vez terminado de bretear le dan 'npm run test' ahi le va a salir el poco de caballadas que hicieron para que las corrigan y este el código limpio.


Luego lo mismo de siempre para hacer el pull request.
